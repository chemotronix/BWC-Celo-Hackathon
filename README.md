# Chevtrox-Celo-Toucan-filecoin for Carbon Credits

# To test the ui

### Clone the application

```sh
 $ git clone https://github.com/chemotronix/BWC-Celo-Hackathon.git
```
### Go to the ui's directory 
```sh
 $ cd ui
```

### Install Dependencis 

```sh
 $ npm install
```

### Run the UI 

```sh
 $ npm run dev
```
